# Data Burger 🍔

Sistema fullstack de gestão e pedidos para hamburguerias, focado em performance e experiência do usuário.

## Tecnologias 🛠️
- **Frontend**: React 19, Vite, Tailwind CSS v4.
- **Backend**: FastAPI (Python), SQLAlchemy.
- **Banco de Dados**: PostgreSQL.

## Como Executar 🚀

### 1. Backend
1. Navegue até a pasta: `cd backend`
2. Ative seu ambiente virtual.
3. Instale as dependências: `pip install fastapi uvicorn sqlalchemy psycopg2-binary python-dotenv`
4. Configure sua `DATABASE_URL` (pode usar um arquivo `.env`).
5. Inicie: `uvicorn main:app --reload`

### 2. Frontend
1. Navegue até a pasta: `cd frontend`
2. Instale: `npm install`
3. Inicie: `npm run dev`
